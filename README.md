# 🎂 DOLCI Cake Ordering and Management System (Admin Interface Prototype)

## Overview

The **DOLCI Cake Ordering and Management System** is an academic team project that was first proposed in **Information Management 1 (IM 1)** as a system concept for a real small cake business, DOLCI. In **Systems Analysis and Design (SAaD)**, one of my teammates from IM 1 and I decided to continue working on the same project, and our two other group members agreed. We expanded the proposal into a comprehensive systems analysis and design project by preparing the Software Requirements Specification (SRS), system models and diagrams, and an **HTML prototype of the administrative interface**. This repository contains the **admin-side user interface prototype** developed as part of the project. The project focused on systems analysis, design, and front-end prototyping and did not include backend or database implementation.

## Repository Scope

This repository includes:
- HTML prototype of the admin interface.
- Project screenshots.
- Selected systems analysis and design documentation.
- Supporting files related to the administrative module.
