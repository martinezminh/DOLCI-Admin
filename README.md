# 🎂 DOLCI Cake Ordering and Management System

## Overview

The **DOLCI Cake Ordering and Management System** is an academic team project that was first proposed in **Information Management 1 (IM 1)** as a system concept for a real small cake business, DOLCI. In **Systems Analysis and Design (SAAD)**, one of my teammates from IM 1 and I decided to continue working on the same project, and our two other group members agreed. We expanded the proposal into a comprehensive systems analysis and design project by preparing the Software Requirements Specification (SRS), system models and diagrams, and an HTML/CSS user interface prototype to demonstrate the proposed system. The project focused on systems analysis, design, and front-end prototyping and did not include database implementation or MySQL.

## Highlights

- First proposed in **Information Management 1 (IM 1)**.
- Expanded into a **Systems Analysis and Design (SAAD)** project.
- Based on the business processes of a real small cake business.
- Included requirements analysis, system modeling, and design documentation.
- Featured an HTML/CSS user interface prototype.
- Completed as a collaborative academic team project.
